# Interference :

When two light waves superimposed, then the resultant intensity will be greater less than that of a single wave. The modification of light energy due to superposition of way is called interference of light.

- **Coherent Source:** Waves having the **same frequency** and **a constant phase difference.**
- **Monochromatic Light:** Light having the **same wavelength** or **same frequency.**
- **Fringe Width:** The distance between two consecutive bright or two consecutive dark width is called fringe width.

### Finding the equation of Fringe width
![[Screenshot 2025-10-16 at 3.22.03 PM.png|400]]
Here,
	$OQ=OR=\frac{d}{2}$
	$PQ=x - \frac{d}{2}$
	$PR=x + \frac{d}{2}$

Now, $$BP^2 = BR^2 + PR^2= (x + \frac{d}{2})^2 + D^2.....(i)$$
$$AP^2=PQ^2+AQ^2= (x- \frac{d}{2})^2 + D^2.....(ii)$$
then, $i^2 - ii^2 \Rightarrow$
$$BP^2 - AP^2 = 2 \cdot 2x\frac{d}{2} = 2xd$$
$$\Rightarrow (BP-AP)(BP+AP)=2xd$$
$$\Rightarrow BP-AP=\frac{2xd}{BP+AP}$$
$$\Rightarrow \Delta x =\frac{2xd}{2D}$$
$$\therefore \ \Delta x = \frac{xd}{D}$$
We can write,
- For Bright Fringe:
$$\Delta x = \frac{xd}{D} = n\lambda$$
$$\Rightarrow x = \frac{n\lambda D}{d}$$
- For Dark Fringe:
$$\Delta x = \frac{xd}{D} = (2n+1)\frac{\lambda}{2}$$
$$\Rightarrow x = (2n+1)\frac{\frac{\lambda}{2}D}{d} = \frac{(2n+1)\cdot \lambda D}{2d}$$
Now, $$\text{Fringe Width = }x_2 - x_1 = \frac{\lambda D}{d}$$

# Interference due to Thin Film

![[Screenshot 2025-10-16 at 3.53.27 PM.png|500]]
#### For Reflected light
$\text{Geometrical Path = Actual path the light travels}$
$\text{Optical Path = Actual path}\times \mu$  
Now,$$\text{Path Difference between BC \& EF} = \mu(BD+DE) - BH$$
$GD = t;\ BD = DE$
$BD+DE=2BD$
in $\Delta BHE,$ 
$$sin\ i = \frac{BH}{BE} \Rightarrow BH = 2BGsin\ i$$
$$\Rightarrow BH = 2BG (sin\ r)\mu\quad [\mu = \frac{sin\ i}{sin \ r}]$$
in $\Delta BGD,$
- $\frac{BG}{GD} = tan\ r \Rightarrow BG = tan\ r \cdot GD = tan\ r \cdot t$ 
- $cos\ r = \frac{GD}{BD} \Rightarrow BD = \frac{t}{cos\ r}$ 

Again,$$BH = 2 \cdot tan\ r \cdot t \cdot sin\ r \cdot \mu =2\mu t \frac{sin^2\ r}{cos\ r}$$
Now, $$\Delta x = 2\mu BD - BH \quad [BD = BE]$$$$=\frac{2\mu t}{cos\ r} - 2\mu t \frac{sin^2r}{cos\ r}$$$$=2\mu t \frac{1- sin^2r}{cos\ r}$$$$=2\mu t cos\ r$$
$$\therefore \Delta x = 2\mu t\ cos\ r \pm \frac{\lambda}{2} \quad [\text{according to stock's law}]$$
For bright fringe, $\Delta x = n \lambda$ 
$$2\mu t\ cos\ r \pm \frac{\lambda}{2} \ n\lambda$$
$$\Rightarrow 2\mu t cos\ r = (n\lambda \pm \frac{\lambda}{2})$$
$$\therefore 2\mu t\ cos\ r = (2n +1) \frac{\lambda}{2}$$
For dark fringe, $\Delta x = (2n+ 1) \frac{\lambda}{2}$
$$2\mu t\ cos\ r \pm \frac{\lambda}{2} = (2n+1)\frac{\lambda}{2}$$
$$\therefore 2\mu t\ cos\ r=n\lambda$$

#### For Transmitted light
$$\Delta x = 2\mu t\ cos\ r \quad [\text{No phase difference. So }\frac{\lambda}{2} \text{ not needed.}]$$
For bright fringe, $$2 \mu t cos\ r = n\lambda$$
For dark fringe, $$2 \mu t cos\ r = (2n+1)\frac{\lambda}{2}$$

### Why the center ring is dark??
At the point of contact, the size of air thin film is $0$. $[t=0]$, $$\Delta x = 2 \mu t cos\ r \pm \frac{\lambda}{2} = \frac{\lambda}{2} \rightarrow \text{dark}$$
![[Screenshot 2025-10-16 at 5.37.03 PM.png|300]]
Here,
- $OC = R$
- $MQ=t=Op$
- $CP=OC-OP=R-t$
- $OQ = r_n$
Now, $\Delta PCM$,
$$R^2 = (R -t)^2 +r_n^2$$
$$\Rightarrow r_n^2 = -R^2 + 2Rt - t^2 + R^2$$
$$\Rightarrow r_n^2 = 2Rt - t^2$$
$$\Rightarrow \frac{D_n^2}{4} = 2Rt \quad [\text{r>>t}]$$
$$\Rightarrow t = \frac{D_n^2}{8R}$$
**for bright:**
$$2\mu t cos\ r = (2n + 1)\frac{\lambda}{2}$$
$$\Rightarrow 2t = (2n + 1)\frac{\lambda}{2}\quad [\text{air medium = 1; r = 0,cos0 = 1}]$$
$$\Rightarrow D_n^2 = 4R(2n + 1)\frac{\lambda}{2}$$
$$\Rightarrow D_{n+p}^2=4R(2(n+p)+1)\frac{\lambda}{2}.....(i)$$
**for dark:**
$$2\mu t cos \ r = m \lambda$$
$$\Rightarrow D_n^2 = 4R \lambda m$$
$$\Rightarrow D_{m+p}^2 = 4R\lambda (m+ p).......(ii)$$

$i^2 - ii^2 \Rightarrow$
$$\lambda = \frac{D_{n+p}^2 - D_n^2}{4PR}$$
**For liquid instead of air:**
for dark: $$D_{m+p}^2 = \frac{4R(m+p)}{\mu}$$
$$\therefore D_{m+p}^2 - D_m^2=\frac{4R\lambda p}{\mu}$$
Now the Refracting index will be:
$$\therefore \mu = \frac{[D_{m+p}^2 - D_m^2]_{air}}{[D_{m+p}^2 - D_m^2]_{liquid}}$$
